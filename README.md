# 🚗 고속도로 교통사고 대시보드

한국도로공사 고속도로 교통사고 상세현황 데이터를 기반으로 제작한  
**Streamlit 웹 대시보드**입니다.  
사고 발생 현황, 노선별 분석, 사고유형 분석, 피해 현황을 한눈에 확인할 수 있습니다.

| 항목 | 내용 |
|------|------|

## 👥 팀원 소개

| 학번 | 이름 | 역할 |
|------|------|------|
| **20101** | 권우진 | 데이터 수집 및 탐색, 페이지 구상 |
| **20502** | 김경민 | AI 활용 및 코드 개발, 페이지 구상 |

### 🔍 역할 상세

**권우진 (20101)**
- 공공데이터포털에서 분석에 적합한 데이터 탐색 및 수집
- 대시보드 전체 페이지 레이아웃 및 구성 기획
- 어떤 데이터를 어떻게 시각화할지 방향 설정

**김경민 (20502)**
- AI 도구를 활용한 데이터 분석 코드 작성 및 개발
- Streamlit 기반 대시보드 구현
- 페이지 구성 기획 및 시각화 차트 설계
