# 🚗 고속도로 교통사고 대시보드

한국도로공사 고속도로 교통사고 상세현황 데이터를 기반으로 제작한  
**Streamlit 웹 대시보드**입니다.  
사고 발생 현황, 노선별 분석, 사고유형 분석, 피해 현황을 한눈에 확인할 수 있습니다.
